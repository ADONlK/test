# test
На вход подаются строки, каждая из которых содержит выражение в постфиксной нотации (операторы идут после операндов). Программа должна вычислить результат каждого выражения и вывести исходное выражение вместе с его значением.

Пример из файла:

3 2 - 1 + вычисляется как (3 - 2) + 1 = 2
2 12 6 / - вычисляется как 2 - (12 / 6) = 0
Требования к программе:

Чтение строк из входного файла.
Парсинг и вычисление каждого выражения с использованием стека.
Запись результатов в выходной файл в формате:
<выражение> = <результат>
